# CFLAG-DB
This repository contains the implementation of CFLAG-BD, a novel Federated Learning (FL) framework
that integrates:

    ⚙️ Hybrid Client Clustering

    ⛓️ Blockchain-Based Model Weight Deduplication

    🧠 Dynamic Learning Rate Schedules

    💾 Memory-Efficient Simulation with up to 10,000 Clients
